# Dashboard

## Tools Used

- **Angular**
- **TailwindCSS**
- **HTML**
- **CSS**

## Installation

- Fork the repo
- Clone repo
- Run `Npm install` to install dependencies
- Run `ng serve` for a local dev server
- Run `ng build` to build the project
